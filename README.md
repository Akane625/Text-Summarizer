# Text-Summarizer
Dataset from [AsadMahmood (Kaggle)](https://www.kaggle.com/datasets/asad1m9a9h6mood/news-articles)

This repository contains a text summarizer using NLP techniques like Word Count Importance, TF-IDF, and TF-IDF + Cosine Similarities. There will be two files, one is the jupyter notebook playground where the project will be tested and created and the other will be the code for the deployment build either streamlit or a website (using react or html/css/js).
